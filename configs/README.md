# Configs

Use this folder for small, named experiment configs.

Examples:

- `tokenizer_tiny.yaml`
- `lm_tiny_cpu.yaml`
- `lm_small_gpu.yaml`
- `rag_tutor_mvp.yaml`
- `evaluation/kinyarwanda_benchmarks.json`
- `evaluation/gemma4_bakeoff.json`

Do not hide important experiment settings inside notebooks.
